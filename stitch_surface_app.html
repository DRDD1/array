Surface Stitching Tool – User Guide

Purpose

This tool allows two regular-grid 3D surface measurements to be aligned and stitched together. The surfaces are loaded from XYZ files and aligned using a lateral shift (X,Y) and an optional vertical offset (dz). The overlap region is used to estimate the best vertical alignment and to compute an RMS residual error.

Basic Workflow

  1. Load Surface A.
  2. Load Surface B.
  3. Use Display Shift X and Display Shift Y to move Surface B relative to Surface A.
  4. Enable Automatic dz to estimate vertical alignment from the overlap region.
  5. Click Refine Locally to automatically search nearby shifts and minimise RMS error.
  6. Inspect the stitched surface and the residual map.
  7. Use Export to save stitched surface and residual maps as XYZ files.

Plots

  • Surface A – original surface A
  • Surface B – original surface B
  • Stitched Preview – combined surface using current alignment
  • Residual Map – difference between A and B in the overlap region

Input File Format

The tool expects a regular-grid XYZ file with three columns: 

X   Y   Z

Each row represents a measured point on the surface. The grid spacing must be constant in both X and Y directions. Comment lines starting with # are ignored.

Example Input File (.xyz)

# Regular-grid XYZ test surfaces for stitching
# spacing_x = 0.1
# spacing_y = 0.1
# Each file starts at X=0, Y=0
# Surface B contains a small Z offset / tilt / noise relative to A in the overlap region

0.000000 0.000000 0.002518
0.100000 0.000000 -0.000459
0.200000 0.000000 0.009323
0.300000 0.000000 0.012820
0.400000 0.000000 0.003928
0.500000 0.000000 0.008905
...

Important Assumptions

  • Both surfaces must be sampled on a regular grid.
  • Grid spacing should be identical for Surface A and Surface B.
  • The tool reconstructs the grid automatically from X and Y coordinates.
  • The stitched surface uses the physical coordinate system of Surface A.
